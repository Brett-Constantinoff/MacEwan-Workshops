# OpenGL-Workshops
OpenGL code for MacEwan Computer Science Students

## How to run
Clone using 
```
git clone --recursive
```
### Mac
Navigate the workshop you would like to run and type 
```
../../bin/premake/mac/premake5 gmake
```

### windows
Navigate the workshop you would like to run and type 
```
..\..\bin\premake\win\premake5.exe vs2019
```

Then build the resulting sln file
